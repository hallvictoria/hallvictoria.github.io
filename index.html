Memo-RY Github

Spring 2020 WiCS Hacks Project
With over 50 million people suffering from dementia and Alzheimer’s disease, my group and I decided to focus our WiCS hackathon project on helping people dealing with this.

We created an iOS and Android mobile application with 3 main features: a to-do list, which displays a user’s day to day activities, memory flash cards for people, places, and events, so that the users can record information and review it later, and an emergency call button. We built the application using React Native and JavaScript.
