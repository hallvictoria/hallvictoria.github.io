Neighborly

Summer 2021 State Farm Hack Day Project
My team's goal for this project was to reduce social isolation, build relationships, and foster communication across State Farm as employees transition to in-person working. To achieve this, we created a social connection platform that functions using gamified challenges and rewards.

We first designed our app collaboratively in Figma, and then we built a mobile-first web app using React. We incorporated Lottie animations and scss for additional styling and included routing between pages for dynamic component navigation. In addition, we added Firebase integration for personal data management and utilized an image recognition model powered by TensorFlow for challenge verification.

Purrdictions

Summer 2021 Project
Being a frequent animal shelter volunteer, I'm no stranger to seeing the same animals still present week after week, and I wanted to identify a pattern and help them get adopted quicker.

I developed an application that would predict how long it would take for an animal to be adopted based on a variety of factors. I used a N Nearest Neighbor algorithm, and I connected the algorithm to a React front end for easier accessibility.

Just Housing

Fall 2020 Hack-TX Project
My team and I wanted to help prevent the real estate industry from taking advantage of customers, so we created a web application to summarize housing contracts and determine if the pricing is fair.

We used the PyZillow API to scrape housing information and a machine learning algorithm to predict a fair price. Furthermore, we implemented an NLP model to recognize important information in the contract and summarize it into easy-to-read terms. Our front end was built using Flask and HTML / CSS, and our backend was implemented with Python.

Memo-RY

Spring 2020 WiCS Hacks Project
With over 50 million people suffering from dementia and Alzheimer’s disease, my group and I decided to focus our WiCS hackathon project on helping people dealing with this.

We created an iOS and Android mobile application with 3 main features: a to-do list, which displays a user’s day to day activities, memory flash cards for people, places, and events, so that the users can record information and review it later, and an emergency call button. We built the application using React Native and JavaScript.
